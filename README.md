# SMS AI Fraud Detection

## Problem
Many users receive fraud SMS messages such as fake bank alerts, OTP scams, and prize messages.

## Solution
This project detects spam and fraudulent SMS using a keyword-based AI model written in C.  
The model analyzes message text and predicts whether the message is safe or fraudulent.

## Architecture
User SMS → AI Detection Model → Fraud/Safe Output

## Technologies Used
- C Programming
- Dataset based training
- Text processing
- GitHub for version control

## Files
- sms_ai.c → AI detection program
- sms_dataset.txt → training dataset
- sms_ai.exe → compiled executable
